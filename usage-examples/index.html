    Usage Examples

    strong.valid.simple Application Configuration

    All the samples on this page are in the AndHow Samples project, which you can checkout from GitHub and run locally. All samples have associated tests that highlight testing techniques.

    Usage Examples

    • An application that starts up and loads property values from a main method . This would be a good template to follow for an executable jar.
    • Load property values from a file on the filesystem. Loading from the filesystem is slightly trickier than loading from the classpath because the filesystem varies between environments.
    • Working with legacy/existing applications that have existing property names. Its easy to add AndHow into an existing application without having to modify existing code.
